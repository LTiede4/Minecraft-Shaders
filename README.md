# Minecraft-Shaders
Compatible for 1.21.11

*(an incompatibility may exist with `Bare Bones Connected Glass`, but it works fine)*

# First, download and install the following:

### [JDK 21 for Windows](https://www.oracle.com/se/java/technologies/downloads/#jdk21-windows)
* Click the `x64 Installer` download link
* Run the installer once it downloads
* Follow the install process

### [Jarfix](https://johann.loefflmann.net/en/software/jarfix/index.html)
* Click the `jarfix.exe` download link
* Run the installer once it downloads
* Follow the install process
